路旁的結緣桌：一位佛教徒看媽祖遶境

文/林吉祥

上週，四月一個和暖的早晨，我前往彰化花壇，加入了媽祖遶境的一段路程。這趟行程是幾週前就說好的，發起的是兩位定居台中多年的外國朋友——一位加拿大人、一位美國人。他們決心走完九天的全程，事先就跟我說好，等他們走到彰化的時候，希望我能出來路上和他們會合。後來另一位美國朋友邀我陪她走一段，我答應了。一方面是想自己親身走一段，一方面也是想守住跟那兩位從台中走過來的朋友之間的約定。

橫跨幾個縣市的遶境

大甲媽祖遶境是世界上規模最大的宗教遊行之一。每年春天，數十萬人徒步走九天，穿過台灣中部的城鎮與稻田，跟在媽祖的神轎之後——媽祖是這座島上深受敬愛的海神，是沿海地區子民的守護神。整段路線幾乎橫跨整個台灣中段，從台中大甲出發，一路向南經過彰化、雲林、嘉義，再原路折返。

要理解這場遶境，得先知道媽祖在台灣人日常生活中是什麼樣的存在。她是台灣最廣為敬奉的神祇之一，許多城鎮裡的媽祖廟都是當地熟悉的地標，特別是西部平原一帶的老聚落。一年一度的遶境，就是這份平日累積、長達一整年的虔誠走出廟門、踏上大馬路的那一刻。

我們一路所見

沿著整條路線，許多人家紛紛在路邊擺起小小的結緣桌。多數提供的是瓶裝水，也有人遞出紙杯裝的麥茶、熱騰騰的包子、或新鮮的麵包。這些自發前來的志工，許多人在隊伍預計經過前好幾個小時就開車到路邊，把車停在路肩，靜靜等到香客走過來。在路上看起來輕而易舉的款待，背後堆疊的是好幾個小時的準備和等待。

沒有人問我們是誰，也沒有人問我們信什麼。他們只是把食物和飲料塞進我們手裡，露出笑容，輕聲說一句：「加油。」

我的加拿大朋友跟我說了一段讓我一直放在心上的話。他說，身為一個西方人，他通常面對大型人潮會感到不安——腦海某個角落，總有一種隱隱的預感，覺得秩序可能會崩壞、暴力可能會爆發。但在這條路上，眼前綿延著那麼多徒步的人，那種預感卻完全消失了。他完全感覺不到那種緊繃。他感受到的，反而是滿滿的善意。素昧平生的人把食物和飲料塞進他手裡，甚至有人直接打開家門——讓他洗個澡、睡一覺、上桌吃一頓飯。他不過是個從門前經過的陌生人，這些人卻把他當家人對待。他平實地告訴我，他真的很感動。

佛陀所教的第一種美德

在佛教的傳統裡，我在那條路上看見的有個名字—— Dāna ，也就是「布施」：自由地給予、不求回報的修行。佛陀教導，布施是修行路上的第一種美德，而給予的形式有三種：

• 財施 ——包括外財（食物、飲水、金錢、物資）與內財（自己的時間與體力）。

財施 ——包括外財（食物、飲水、金錢、物資）與內財（自己的時間與體力）。

• 法施 ——分享知識、技能，或能幫助他人面對人生的觀點與智慧。

法施 ——分享知識、技能，或能幫助他人面對人生的觀點與智慧。

• 無畏施 ——對焦慮或孤單的人，給予安心、溫暖與鼓勵。

無畏施 ——對焦慮或孤單的人，給予安心、溫暖與鼓勵。

走一趟媽祖遶境，你會親眼看見這三種布施被一般人實踐著。

擺桌、送食物的人在做財施——他們付出的是準備的開銷，也是在路邊等候的那好幾個小時。沿路幫忙引導的人在做法施——指點大家去休息站、回答各種問題、跟第一次參加的人解釋這些儀式。而那些在香客經過時喊著「加油」的人，則是在做無畏施。腳上起水泡的香客，因為那一聲，能再多走十步。

恭敬心：修行的土壤

這條路上不只有布施。還有恭敬心——而恭敬心，以它自己安靜的方式，對佛法之道來說同樣重要。

台灣人對媽祖的虔誠，是好幾個世代累積下來的。那是好幾百年來在媽祖庇佑下生活的人家所累積的信賴——一群願意頂著大太陽走九天、願意為陌生人擺桌相迎、願意在比自己更大的存在面前低頭的人。這種能夠跨越世代、無論順境逆境都持續而真誠地恭敬的能力，本身就是一種力量。

當我看見一群已經懂得恭敬神聖事物、已經懂得不計較地付出、已經懂得歸屬於比自己更大事物的人——我看到的並不是與佛法相違的東西。我看到的，是一塊已經被靜靜耕耘了好幾世的土壤。

這條路真正在教我們的事

走完這一段媽祖遶境的路，我帶回家的領悟其實很簡單。我看見了兩種美德，這兩種美德也正是我們佛教徒所重視的：布施與恭敬心。這兩種美德並不是我們佛教徒獨有的——它們是大家共通的價值。布施——自由地給予、不求回報——沿途到處都看得見：那一雙雙擺起桌子、把食物與水送出來給陌生人的手。恭敬心——願意在比自己更大的存在面前低頭——展現在那些年復一年願意走九天的人身上，這份虔誠是代代相傳下來的。

當這兩種美德在整個社會中交織在一起，它們成就出來的，是一個更溫柔、更祥和的台灣——在這裡，陌生人會關懷陌生人，人們願意彼此幫忙。這是值得珍惜的——對任何社會、對任何傳統來說，都是。那正是我在媽祖遶境路上感受到的同一份溫柔；而正是這份溫柔，放大到整座島嶼，讓台灣成為一個讓陌生人都覺得被歡迎的地方。
