俱舍論隨眠品導讀（品五）

六隨眠展開為九十八隨眠的完整分類系統，以及隨眠與纏、漏、瀑流等異名的辨析——俱舍論解釋輪迴根本原因的核心章節。

總覽：本品在《俱舍論》九品中的位置

品1～2 界品／根品器世間與有情的基本架構
→
品3～4 世間品／業品有情如何流轉、業如何招感
→
品5 分別隨眠品業從何而來——惑的體系
→
品6 分別賢聖品如何斷惑證果

品四已說業如何招感異熟果，本品開篇第一句便追問：業由何而生？答案是「隨眠」——貪、瞋、慢、無明、見、疑等潛伏的煩惱。全論到此正式從「果」（世間、業）回溯到「因」（惑），為下一品「如何斷惑證得賢聖果位」鋪路。本品橫跨卷十九至卷二十一，是《俱舍論》全論篇幅最長、法相最綿密的一品，前後又分「之一、之二、之三」三段展開。

「隨眠是諸有本，故業離此無感有能。」——業若沒有隨眠的滋養，便無感生後有之力；本品要做的，正是把這個「有本」的隨眠，從六種、七種、十種一路開展到九十八種，再逐一分辨其體性、所緣、繫縛與斷除之道。

科判骨架：品五如何開展

分別隨眠品第五（卷十九～卷二十一，全論九品中第五品）
  • 之一（卷十九）：隨眠的體性與分類
    • 隨眠為諸有之本——現起能為十種事，令不能越自界地
    • 六隨眠、七隨眠、十隨眠、九十八隨眠層層開展
    • 隨眠是否心相應——有部（即煩惱體）與經部（煩惱種子）之諍
    • 見所斷／修所斷，遍行隨眠，緣有漏／緣無漏，所緣隨增／相應隨增
    • 五見自體、四顛倒、七慢與九慢類
    • 不善根、無記根、四種記（一向、分別、反詰、捨置）
  • 之二（卷二十）：隨眠如何繫縛三世
    • 自相惑（貪瞋慢）與共相惑（見疑癡）如何繫縛過去現在未來
    • 三世實有——毘婆沙師立宗與教理二證
    • 法救、妙音、世友、覺天四家釋三世，及經部層層破斥
    • 斷與離繫的差別、十六法十六識所行境
    • 十隨眠生起次第（無明疑邪見身見邊見戒取見取貪慢瞋）、惑起三因緣
    • 隨眠異名之一：漏、瀑流、軛、取
  • 之三（卷二十一）：隨眠異名與斷惑歷程
    • 隨眠等異名之義：微細、二隨增、隨逐、住流漂合執
    • 結（九）、縛（三）、隨煩惱、纏（十）——五種異門總說
    • 五順下分結、五順上分結；六煩惱垢；五蓋
    • 見修所斷、善惡無記、界繫、依何識起、與受相應等分別
    • 斷惑四因、四種對治、四種遠性
    • 九遍知——所依道果、六時重得、誰成就幾種

隨眠的層層展開：六、七、十、九十八

本品開篇先問「隨眠有幾」，論主不是一次給出最終答案，而是示範一種漸次展開的分析方法——同一批煩惱，換一個角度切入就會呈現不同的數目，數目不同不代表彼此矛盾，而是分類角度不同。

六隨眠貪、瞋、慢、無明、見、疑
→
七隨眠貪開為欲貪、有貪（色無色界貪）
→
十隨眠六中「見」再開為五：身見邊見邪見見取戒取
→
九十八隨眠十隨眠 × 五部（四諦所斷＋修所斷）× 三界

貪為何要分為欲貪、有貪二種？論中說是因為色無色界的貪多「於內門轉」（緣自身定境而非外境），又為遮止修定者誤將上二界的禪定境界當作真解脫，特立「有貪」之名以顯其仍非真淨。「見」為何要開為五？因為身見、邊見、邪見、見取、戒禁取雖同屬「見」，行相與所斷部類皆不同，若籠統合一，後面九十八隨眠的分部便無法安立。

界五部隨眠數（見苦／見集／見滅／見道／修所斷）小計
欲界十、七、七、八、四36
色界五部同上，惟各部除瞋31
無色界五部同上，惟各部除瞋31

十隨眠中，身見、邊見只在見苦所斷一部才有；戒禁取通見苦、見道二部；邪見、見取、疑通見苦集滅道四部；貪、瞋、慢、無明則通五部（四諦所斷加修所斷）。色界、無色界沒有瞋（瞋唯繫屬欲界，因上二界無苦異熟果可資逼惱），故二界各部只需在欲界基礎上除瞋，因而各少五個，成三十一，合計三十六加三十一加三十一，正是九十八。

隨眠異名：一堆煩惱、多種稱呼

《俱舍論》對同一批煩惱，還依不同作用面向另立五種異名——漏、瀑流、軛、取（卷二十）與結、縛、隨煩惱、纏（卷二十一）。理解這些異名，能幫助讀者在其他經論中遇到「四流」「四取」「九結」等詞彙時，知道它們與隨眠的關係。

漏欲漏（欲界煩惱并纏，除癡，共四十一物）、有漏（色無色界煩惱除癡，共五十二物，因二界同屬無記、於內門轉、依定地而生，三義相同故合為一漏）、無明漏（三界十五種無明，因無明是諸有之本，故別立一漏）。
瀑流、軛體性與漏相同，但另從欲漏、有漏中析出「見」，別立見瀑流、見軛——因見的行相猛利，不順於漏「令心住著」之義，須與貪瞋等合說才可立漏名。
取欲取（欲軛并欲界無明）、見取（見軛中除去戒禁取）、戒禁取（獨立為一取，因它專誑在家出家二眾，於聖道為怨）、我語取（有軛并色無色界無明，因耽著自體而非味著外境，故稱「我語」）。
結、縛、隨煩惱、纏結有九種（愛恚慢無明見取疑嫉慳）；縛只取貪瞋癡三種（依樂苦捨三受安立）；隨煩惱泛指一切隨心而起的染污心所；纏另立十種（無慚無愧嫉慳悔眠掉舉惛沈，毘婆沙師加忿、覆共十）。
五順下分結

有身見、戒禁取、疑、欲貪、瞋恚——順益欲界（下分）。前三令異生無法超出欲界，後二縱使已超出欲界，仍會被拉回，如守獄卒防邏人；預流果斷前三結，不還果則五結俱斷。

五順上分結

色貪、無色貪、掉舉、慢、無明——順益色無色界（上分）。未斷此五，縱使已離欲界染，仍不能超出上二界；唯阿羅漢方能究竟斷盡。

深入辨析：三世實有之諍——隨眠如何繫縛過去未來？

卷二十開頭先問「隨眠能繫何事」：貪瞋慢等「自相惑」與見疑癡等「共相惑」，各自如何繫縛過去、現在、未來三世的所緣境。既然隨眠可以繫縛「已謝」的過去與「未生」的未來，論主便順勢追問：過去、未來到底是有還是無？由此引出說一切有部教義史上最重要的一場辯論，也是「說一切有部」得名的由來。

毘婆沙師（有部）：三世實有

頌曰：「三世有由說，二有境果故，說三世有故，許說一切有。」論主舉教證（世尊說「若過去色非有，不應多聞聖弟子眾於過去色勤脩厭捨」）與理證（識必託境方生，若無過去未來境，識應闕緣而不得生）證成三世實有。凡主張三世皆定實有者，方許為「說一切有部」，此宗因此得名。

世友尊者：由「位」立世，約作用最善

法救主張由「類」不同、妙音主張由「相」不同、世友主張由「位」不同、覺天主張由「待」不同，四家皆許法體恒有，只是所立三世的道理各異。論主評判前三家皆有過失，唯世友「由位不同，約作用立世」最為周延——法有作用時名現在，作用未起名未來，作用已滅名過去，法體本身不隨作用起滅而變異。

經部的層層反詰

論主隨即以經部師之說反覆詰難「法體恒有、但由作用分三世」的立場：若法體恒常不變，何以能令作用有時而起、有時而滅？若說作用本身還要再分過去現在未來，便成無窮之過。經部主張，所謂「三世實有」的教證，其實只是「曾有」（過去）、「當有」（未來）的因果假立，並非法體如現在般真實存在；並引《勝義空經》「眼根生位無所從來，滅時無所造集，本無今有、有已還無」，作為對法體恒有說的直接否定。毘婆沙師最後只能承認「法性甚深，非尋思境」。

這場辯論不只是抽象的形上學遊戲，它直接關係到本品的主軸——隨眠如何繫縛未斷之事。若過去、未來的煩惱與所緣境並非實有，隨眠的繫縛作用要安立在何處？有部與經部由此分道，也與前段「隨眠是心相應法（有部）或是不現行的煩惱種子（經部）」之諍一脈相承。世親雖依有部宗造《俱舍論》，行文中卻屢次徵引經部觀點層層詰難有部立場，這正是本論素有「以有部宗而寓經部義」之稱的一大特色，在本品表現得尤其鮮明。

讀書會抓法

  1. 先掌握六→七→十→九十八的展開邏輯。
    不必死背九十八個名相，而是理解每一次「開展」背後的理由（貪為何分二、見為何分五、部界如何相乘），數字自然就能推導出來。
  2. 分清「隨眠是心相應法」還是「不現行的種子」。
    有部主張隨眠即煩惱體、是相應法；經部主張隨眠只是煩惱的種子功能，非相應非不相應。這關係到「斷惑」到底斷的是什麼，日後也是唯識「種子」說的重要伏筆。
  3. 留意三世實有之諍如何從「隨眠繫縛三世」自然引出。
    它不是插話，而是回答「隨眠如何繫縛已謝的過去、未生的未來」這個實際問題所需的形上學基礎。
  4. 用五順下分結、五順上分結對照四向四果。
    預流斷前三順下分結，不還斷盡五順下分結，阿羅漢才究竟斷五順上分結——本品的法相分類最終要落實到修證次第上。
  5. 記住斷惑四因與四種對治的差別。
    見所斷惑由前三因（遍知所緣、斷能緣、斷所緣）決定，一見道即頓斷；修所斷惑則唯由第四因「對治起故斷」，須漸修漸斷，二者修行下手處並不相同。
讀書會提問：
  • 經部主張隨眠「非心相應非不相應」，是煩惱睡位的種子功能；這跟有部主張隨眠即是相應法的說法，對我們實際觀察自己煩惱起伏的方式，會有什麼不同的下手處？
  • 三世實有與經部破斥，這場千餘年前的哲學辯論，對我們今天理解「業從過去如何感未來果」有什麼啟發？
  • 九十八隨眠中，色、無色界比欲界各少了「瞋」，這個差異背後反映了什麼樣的因果道理？
  • 五順下分結、五順上分結分別對應解脫道的哪個階段？作為在家共修者，這對我們評估自身修行進度有何提醒？
