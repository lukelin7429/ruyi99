摩訶般若波羅蜜經

鳩摩羅什譯《摩訶般若波羅蜜經》（大品般若經）九十品，先建立九科全經地圖，掌握這部《大智度論》所解釋底本經典的行文脈絡與思想開展。

《摩訶般若波羅蜜經》（又稱「大品般若經」），後秦鳩摩羅什譯，全經二十七卷、九十品，是般若系經典中份量最重、義理最完整的一部，也是龍樹菩薩《大智度論》逐句解釋的底本——羅什大師譯出此經後，隨即又譯出《大智度論》為之作釋，兩部合看，正是理解般若空義最直接的途徑。全經以「序品」佛陀放光集眾、開示「菩薩摩訶薩欲成就一切功德，當學般若波羅蜜」總綱起始，其後透過佛與舍利弗、須菩提等尊者反覆問答，將十八空、六度、三十七道品、十地、五眼、十力四無畏十八不共法等一切大乘法門，收攝於般若空慧一貫之下，最後以「常啼菩薩」求法的動人本生故事與「囑累品」付囑流通作結。本頁先建立九十品的全經地圖，依內容脈絡分為九個大科，幫助讀者掌握整部經的行文次第與思想開展；各品逐品導讀將於後續視需要陸續擴充。

怎麼讀這部經

先看大科結構表，掌握九十品如何分段展開、彼此如何承接，再視自己關心的主題（例如十八空、六度、常啼菩薩本生）選段深入。本經文字反覆、善用重頌與譬喻，初讀者不必逐字通讀，抓住每一大科的核心論點即可。

本站整理原則

本頁的九科分段，是本站依九十品品目內容與行文脈絡自行歸納整理，方便閱讀對照，並非引用某一家古德判教，也不等同於《大智度論》的科判——遇有出入，請以經文本身與《大智度論》的解釋為準。

版本說明

底本：《大正藏》第八冊 No.223《摩訶般若波羅蜜經》，鳩摩羅什譯。網路可另參 CBETA 線上閱讀、維基文庫全文對讀；異譯本有西晉無羅叉《放光般若經》（T221）、竺法護《光讚經》（T222）可互相參照。

與《大智度論》的關係

《大智度論》是龍樹菩薩逐句解釋本經的釋論，鳩摩羅什漢譯本經之後隨即譯出。本站讀書會「般若」分類下已建有《大智度論》卷次導讀（見〈大智度論〉頁面），目前已建的卷一對應本經〈序品第一〉，卷五十四則對應到〈問住品第二十七〉附近，可知大智度論詳細逐句解釋的進度，大致與本經前段品目同步展開。一般學界公認羅什所譯大智度論到後段對本經後面品目的解釋轉趨簡略、節略幅度加大，但確切的節略起點眾說不一，本站不作武斷認定；讀者若想對照兩經進度，可直接查閱大智度論該卷開頭引用的經文段落。

九十品架構：九科全經地圖

科次品目範圍大科標題內容大要
一品1序分序品：六成就、佛放光集十方眾、告舍利弗「菩薩欲成就一切功德當學般若波羅蜜」，反覆列舉般若總攝諸法功德，是全經總綱。
二品2～9總標般若正行奉鉢、習應、往生、歎度、舌相、三假、勸學、集散：從舍利弗、須菩提對答展開，闡明「以不住法住般若」與三假施設，破除諸法實有執。
三品10～24深觀諸法實相相行、幻學、句義、金剛、樂說、辯才、乘乘、莊嚴、問乘（摩訶衍義）、廣乘（四念處等三十七道品）、發趣（十地初說）、出到、勝出、等空、會宗：廣說大乘義理與空有二宗會通。
四品25～37十八空與無生深義十無（十八空廣釋）、無生（無生法忍）、問住（帝釋天主問答）、幻聽、散花、三歎、滅諍、大明（寶塔顯現）、述成、勸持、遣異、尊導、法稱：義理與瑞相交織，並由諸弟子述成勸持。
五品38～52福德校量與魔事考驗法施（十善業道）、隨喜迴向、照明、信毀、歎淨、無作、遍歎（百波羅蜜異名）、聞持、魔事、兩過、佛母（般若為諸佛之母）、問相：以福德校量顯般若殊勝，並教導辨別魔事障道。
六品53～69甚深不退與夢行成辦、譬喻、知識、趣智、大如、不退、堅固、深奧、夢行、河天、不證、夢誓、魔愁、等學、淨願、度空、累教、無盡、攝五（六度相攝）、方便：辨明不退轉相與甚深空義，夢中修行亦不離空。
七品70～78三慧與六度無漏行三慧（聞思修慧攝一切法）、道樹、道行、三善、遍學、三次、一念（無漏一念頓修六度）、六喻（六度如夢如化）、四攝：說明菩薩以三慧統攝萬行，六度化他不離空觀。
八品79～87果地淨土與甚深法義善達、實際、具足、淨土（莊嚴佛國）、畢定（菩薩畢竟不退之定）、差別（會通四諦）、七譬、平等、如化：說明菩薩淨土行與果地功德，會通聲聞四諦與大乘空義。
九品88～90常啼求法本生與流通常啼（薩陀波崙菩薩不惜身命東行求法的著名本生）、法尚（曇無竭菩薩為說般若）、囑累（佛以般若付囑阿難流通）：以感人的求法故事收束全經，並殷重付囑流通。

建議學習路線

初學入門

  • 先讀〈序品〉掌握全經總綱
  • 對照本站《大智度論》卷一導讀
  • 了解九科大意，再挑感興趣段落深入

義理重點

  • 十八空（十無品）
  • 三十七道品（廣乘品）
  • 四諦會通（差別品）

敘事與行持重點

  • 常啼菩薩求法本生（常啼、法尚品）
  • 六度相攝與無漏一念（攝五、一念品）
  • 淨土莊嚴行（淨土品）

品導覽（已上線前二十四品，陸續增建中）

品1
序品第一
世尊放光現瑞、十方雲集後，佛告舍利弗當學般若波羅蜜的無數理由，六度皆以「不可得」為根本，是全經二十七卷的總綱所在。

 品2
奉鉢品第二
舍利弗兩問，佛陀兩答：菩薩不必出家斷欲方得菩提，而行般若波羅蜜的關鍵在不見一切名字，色空不二。
閱讀導讀 →
 品3
習應品第三
佛陀答舍利弗「云何與般若相應」，從五蘊、十八界到七空逐層說空，帶出「色即是空」名句，末以空相應生大慈大悲作結。
閱讀導讀 →
 品4
往生品第四
舍利弗問菩薩死此生彼之處，佛答以方便力自在受生，並廣說五眼六通、身口意不淨即取相、一切法不可得之理。
閱讀導讀 →
 品5
歎度品第五
本品由四大聲聞與四眾共讚般若波羅蜜十六異名，佛陀印可並說明菩薩躬行六度、自利利他，故為人天善趣資具樂具與聖果之根源。
閱讀導讀 →
 品6
舌相品第六
舌相放光召集十方菩薩雲集，十萬億弟子當場得無生法忍並獲授記，以感應事件為般若功德作見證，銜接歎度品與三假品。
閱讀導讀 →
 品7
三假品第七
須菩提問「菩薩」是何法，佛以三假施設與四句都遣層層破執，說明一切法不可得，菩薩因不著而增益六度、得阿惟越致。
閱讀導讀 →
 品8
勸學品第八
從「欲得諸法當學般若」的總持誓願，到辨明法愛之障、十八空無生與心相常淨，指出般若波羅蜜是聲聞、辟支佛、菩薩三乘共學的根本教法。
閱讀導讀 →
 品9
集散品第九
本品是「總標般若正行」大科最後一品：須菩提歷數諸法乃至菩薩之名皆集散不可得，並以不受三昧、先尼梵志之信、舍利弗問答收束全科。
閱讀導讀 →
 品10
相行品第十
本品以「著相／不著相」辨明般若方便之別，用色空不二推展到一切法，並帶出無量三昧法門，讀書會可藉此練習『不可得』的觀法。
閱讀導讀 →
 品11
幻學品第十一
佛藉「幻」直探諸法實相，說色即是幻、幻即是色，以無所得故能學般若得一切智；再以八相配六度說方便，辨析善惡知識之判準在於導向而非對錯。
閱讀導讀 →
 品12
句義品第十二
本品佛陀連用十餘種譬喻，說明菩薩句義如空中鳥跡、夢中所見，畢竟無所有；再列一切法六對範疇，教示不應執取、法法不二相不動。
閱讀導讀 →
 品13
金剛品第十三
佛陀就「何以名摩訶薩」層層解說：於必定眾為上首，發大誓莊嚴、代眾生受苦等不可壞如金剛的大心。
閱讀導讀 →
 品14
樂說品第十四
舍利弗以斷諸見網解說摩訶薩，須菩提續以性空無所得為證，示現連不著本身也不可執著的深觀智慧。
閱讀導讀 →
 品15
辯才品第十五
本品由富樓那彌多羅尼子主答，以六度彼此含攝説明大誓莊嚴，再以四禪四無量心、三十七道品及十八空無所得之智，開展菩薩發趣大乘的修證次第。
閱讀導讀 →
 品16
乘乘品第十六
富樓那答舍利弗：乘六度、觀一切法但有名字不可得，即是乘於大乘；終不離菩薩行，得一切種智轉法輪，十方諸佛稱歎。
閱讀導讀 →
 品17
莊嚴品第十七
從六度化現如幻的莊嚴出發，層層深觀至「非作非不作」「無縛無脫」，揭示菩薩壯闊行持背後的空性根基。
閱讀導讀 →
 品18
問乘品第十八
須菩提問何等是摩訶衍，佛陀以六波羅蜜、十八空、百八三昧三層鋪陳，具體定義大乘的修證內容，是深觀諸法實相的重要一品。
閱讀導讀 →
 品19
廣乘品第十九
本品逐一唱名四念處、五根五力、七覺分、八聖道分、佛十力、十八不共法與陀羅尼字門，以性空「不可得」貫穿三十七道品，構成全經最完整的大乘總說。
閱讀導讀 →
 品20
發趣品第二十
須菩提問大乘發趣，佛陀以十地為綱，逐地列出治地業並一一解釋，每一項都歸結於諸法不可得，展現菩薩行位與空觀不二。
閱讀導讀 →
 品21
出到品第二十一
本品以三問三答開展：大乘從三界出、住無住處薩婆若，而乘、乘者、所乘法皆不可得，正因無所得，菩薩摩訶薩方能乘大乘出三界、住薩婆若。
閱讀導讀 →
 品22
勝出品第二十二
佛答摩訶衍為何勝出世間：從三界五蘊到佛三十二相、法輪、眾生涅槃，一切法皆無法非法，故大乘無所不遍。
閱讀導讀 →
 品23
等空品第二十三
佛陀以虛空為喻，印可摩訶衍與空等、能含受無量眾生、不動不住、三世平等，展開「無所有」連鎖論證，深觀大乘教法自身之空性。
閱讀導讀 →
 品24
會宗品第二十四
富樓那質疑須菩提說摩訶衍偏離般若，佛陀親口印證：六度、三十七道品乃至十八不共法皆攝入般若，與摩訶衍無二無別，一相無相。
