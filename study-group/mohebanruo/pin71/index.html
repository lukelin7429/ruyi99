菩薩無所得而發心度眾，六度成佛樹

摩訶般若波羅蜜經〈道樹品第七十一〉導讀

以種樹喻說明菩薩雖知眾生不可得，仍漸修六度成就一切種智，並辨析諸法無性下方便力與世諦第一義諦不二之理。

總覽：本品在「三慧與六度無漏行」中的位置

大科七（品70～78）以聞思修三慧統攝六度萬行，探討六度如何在無漏、無所得的正見下修習。〈道樹品第七十一〉一開篇，須菩提便接著先前的問答，繼續逼問一個核心矛盾：般若甚深，菩薩明明「不得眾生」，卻仍要為眾生求無上菩提——這豈不是像想在虛空中種樹一樣，根本無處著力？佛陀不迴避這個矛盾，反而用「種樹喻」正面承接，說明菩薩雖不執取眾生實有，仍可漸漸修行六波羅蜜，終能成就利益眾生的「佛樹」。本品由此展開，從種樹喻的具體譬喻，一路推進到「如」貫通凡聖的義理、初發心菩薩的校量福德，最後收束於「一切法無性」與「方便力」的辨析——為大科七後續各品如何在無所得中具體操作六度，先立下一個總綱式的說明。

「譬如人種樹，不識樹根莖枝葉華果而愛護溉灌，漸漸長大華葉果實成就，皆得用之。」——不必先窮盡究竟義理才能下手修行，護持灌溉、漸次而行，佛樹自然華果具足、利益眾生。

敘事骨架：一品六番問答

甲　種樹喻與果地大用
  • 須菩提問：為（不可得的）眾生求無上菩提，如於虛空中種樹，甚難
  • 佛以種樹喻作答：漸漸行六波羅蜜，得一切種智成佛樹，以葉、華、果實利益眾生
乙　菩薩如佛：以「如」貫通凡聖
  • 須菩提說：如此菩薩當知如佛
  • 佛印可，並以「如」字統攝如來、辟支佛、阿羅漢乃至一切有為無為法，說明菩薩學如般若波羅蜜，層層相因終至轉法輪度眾生
丙　校量福德：初發心菩薩的分量
  • 須菩提問：初發意菩薩為眾生求菩提，得幾許福德
  • 佛以聲聞辟支佛意、乾慧地乃至辟支佛地等層層作比，襯出初發意菩薩福德之殊勝，再校量入法位、向佛道、佛功德的次第
丁　一切種智無所有
  • 須菩提問：初發心菩薩當念何法
  • 佛答：當念一切種智——無所有、無念、無生、無示，無法緣、念為增上、寂滅為行、無相為相
戊　諸法無性與方便力
  • 須菩提追問：一切種智乃至一切有為無為法既皆無法（自性空），初發意菩薩如何仍能以方便力行六度、淨佛國土、成就眾生
  • 佛答：知國土眾生亦無性，即是方便力；因緣具足時一念相應慧得一切種智，煩惱習永盡
己　世諦與第一義諦不異
  • 須菩提問：諸法與諸法性既離，如何能分別有無
  • 佛答：菩薩以世諦示眾生若有若無，非以第一義；然世諦如即第一義諦如，二者無異

核心譬喻：種樹喻——六度如何長成佛樹

發心為眾生求菩提眾生實不可得
→
漸漸行六波羅蜜如愛護溉灌
→
得一切種智成佛樹
→
以葉華果實益眾生

佛陀把菩薩比作種樹的人：種樹之初不必先弄懂根莖枝葉華果的每一細節，只要用心愛護灌溉，樹自然漸漸長大、花果成就，人人皆能受用。菩薩修六度也是如此——不必也不可能先把「眾生」執取為實有對象，只要如法漸修，自然成就利益眾生的果報。

葉益眾生因菩薩摩訶薩故，眾生得離三惡道——這是佛樹最初、最基礎的庇蔭。
華益眾生因菩薩故，眾生得生剎利大姓、婆羅門大姓、居士大家，乃至四天王天以上直到非有想非無想處——增上生的果報。
果益眾生菩薩得一切種智，令眾生依三乘法證須陀洹、斯陀含、阿那含、阿羅漢果，乃至辟支佛道、佛道，終於無餘涅槃而般涅槃——究竟解脫的果實。

須菩提隨即讚歎：這樣的菩薩當知如佛。因為正是這樣的因緣，才斷盡地獄種、畜生種、餓鬼種，斷一切諸難、貧窮下賤與三界流轉——若無菩薩發心，世間便無過去未來現在諸佛，也無辟支佛、四果聖者，三惡趣與三界更無斷絕之時。

「如」貫通凡聖：從學如般若到轉法輪的層層相因

佛陀進一步說明：以「如」故說如來，以如故說辟支佛、阿羅漢、一切賢聖，以如故說色乃至識、一切有為無為法——這一切「如」實無差異，故名為如。菩薩學此如而得一切種智，故亦名如來。經文接著用「X已，則Y」的句式，一環扣一環，鋪出菩薩自利利人的完整歷程：

學如般若波羅蜜起點
→
學一切法如
→
具足一切法如
→
住一切法如得自在
↓
善知一切眾生根
→
知眾生根具足
→
知眾生業因緣
→
得願智具足
↓
淨三世慧
→
饒益一切眾生
→
淨佛國土
↓
得一切種智
→
轉法輪
→
安立眾生於三乘入無餘涅槃

這條連環的起點是「學如般若波羅蜜」，終點是「轉法輪、安立眾生於三乘」——中間每一步都是前一步的自然結果，並無跳躍或勉強嫁接。佛陀最後總結：菩薩摩訶薩欲得一切功德、自利利人，應發阿耨多羅三藐三菩提心，正是為此而說。

校量福德：初發心菩薩的分量

須菩提問：初發意菩薩摩訶薩為眾生求菩提，究竟得多少福德？佛陀以層層遞進的校量作答——每一次都是「百倍千倍巨億萬倍，乃至算數、譬喻所不能及」，凸顯發菩提心本身的稀有與力量：

校量對象（福德較少的一方）校量結果
三千大千世界眾生皆發聲聞、辟支佛意不如初發意菩薩——因為發聲聞辟支佛意者，皆因菩薩出世而有；菩薩終不因聲聞辟支佛而出
三千大千世界眾生皆住乾慧地不如初發意菩薩
三千大千世界眾生皆住性地、八人地、見地、薄地、離欲地、已辦地、辟支佛地不如初發意菩薩
初發意菩薩不如入法位菩薩
入法位菩薩不如向佛道菩薩
向佛道菩薩不如佛功德

這一連串校量不是要人比較高下、生起我慢，而是要凸顯：即使只是「初發意」，只要方向對準無上菩提，其力量便已遠遠超越聲聞辟支佛的整個果位體系——因為菩薩之道最終要成就的，是連聲聞辟支佛都仰賴其出世的佛果本身。

深入辨析：無性而能行——方便力的弔詭

須菩提追問一切種智「何等緣、何等增上、何等行、何等相」，佛答：一切種智無法緣、念為增上、寂滅為行、無相為相。須菩提由此擴大範圍：不只一切種智，色受想行識、內外法、四禪四無量、三十七道品、三三昧、八背捨、十力四無所畏、大慈大悲，乃至有為相無為相，是否也都「無法」？佛陀肯定：一切法皆無法——因為一切法因緣和合而生，自性本空，「自性無」即是「無法」。

一、無性不是修行的障礙，而是方便力本身

這就逼出全品最尖銳的問題：若一切法皆無自性，初發意菩薩憑什麼還能以方便力修六度、淨佛國土、成就眾生？

看似矛盾之處

方便力通常被理解為「有主體、有可修之法、有可成就之淨土」才能施設；若連菩薩、六度、國土、眾生本身都無自性，方便從何談起？

經文的回答

菩薩能學諸法無性，亦能淨佛國土、成就眾生——「知國土眾生亦無性」，這個「知」本身就是方便力，不是在無性之外另找一個方便。菩薩行六度修學佛道，同時了知佛道本身也無自性；因緣具足時，以「一念相應慧」得一切種智，一切煩惱習氣永盡，以佛眼觀三千大千世界，「無法尚不可得，何況有法」。

這個原則貫徹到底：菩薩布施時，布施本身、受者、菩薩自己的心，乃至一切種智的能得、所得之法、所得之處，都是「無法尚不可知，何況有法」。理由是——一切法本性如此，不是佛所造作，也不是聲聞辟支佛或任何人所造作，一切法本來就「無作者」。

二、世諦與第一義諦不異

須菩提再問：既然諸法與諸法性本自相離，一個「已離之法」如何還能分別有、無？佛陀答：菩薩是以世諦向眾生示現若有若無，並非依第一義。須菩提追問世諦與第一義諦是否有別，佛陀的回答是全品的收筆：兩者無異——因為世諦的「如」，就是第一義諦的「如」；只因眾生執著五受陰中的相，不知「無所有」，菩薩才借世諦施設有無，目的是引導眾生最終認識那清淨無所有的實相。

讀書會抓法

  1. 種樹喻不是勵志比喻，是修行次第的說明。
    不必先窮盡「眾生是否實有」這類究竟義理才敢下手修行，護持灌溉、漸次而行，佛樹自然華果具足。
  2. 「如」是貫通凡聖的公分母。
    如來、辟支佛、阿羅漢、色受想行識，乃至一切有為無為法，都以「如」而立名，本無差異——這是理解「凡聖不二」的重要線索。
  3. 校量福德的重點在發心，不在比較。
    初發意菩薩的福德遠勝聲聞辟支佛全體果位，說明的是菩提心方向的力量，不是拿來生起優越感的資本。
  4. 「無性」與「方便力」不是兩件事。
    知一切法無自性，這個「知」本身就是方便力所在，不必在空性之外另尋一個能修、能證的實體。
  5. 世諦與第一義諦不異，有無皆是方便施設。
    菩薩說有說無，是為了眾生能懂，最終要引向的是同一個「清淨無所有」。
讀書會提問：
  • 「眾生不可得，而為眾生求阿耨多羅三藐三菩提」，這句話乍看矛盾，你如何理解菩薩「不得眾生實法」卻仍全力度眾生的心態？
  • 經文以「如」字統攝如來、辟支佛、阿羅漢乃至一切有為無為法，這對你理解「眾生皆有佛性」或「凡聖不二」有什麼啟發？
  • 「一切法本性爾，非佛作，非聲聞辟支佛作，亦非餘人作」——若連佛陀都「未曾造作」任何一法，我們平常說的「修行」「證果」到底是在做什麼？
