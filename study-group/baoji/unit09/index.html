出世智藥治導讀：求心了不可得

連醫王耆域都答不出「心中結使煩惱」該用什麼藥治，唯有出世智藥才能根治。這一單元是「正說菩薩道」全篇的收尾——從標舉法性空觀心，到三門推求心相了不可得，再到二十四句膾炙人口的呵心妄有譬喻，最後層層推導顯示無性即是聖性，全篇在「上來，正說菩薩道已究竟」一句中圓滿收束。

整個單元其實是一條完整的推理鏈：從「心到底是什麼」的追問出發，經三門推求證明心了不可得，再從「不可得」一路邏輯推導到「無為」，最後從無為顯示出聖者的體性——這是漢傳佛教「觀心法門」最經典的一段論證，也是全經「正說菩薩道」的壓軸總結。

全經「正說菩薩道」的最後一句：「這就是菩薩的出世智藥。以此自得解脫，也以此解脫眾生……上來，正說菩薩道已究竟。」——本單元讀完，就是全經乙一（正說菩薩道）的完整終點；下一單元將轉入乙二「兼說聲聞道」。

戊一 舉喻起說：連醫王耆域也答不出的心病

佛先以譬喻起說：南閻浮提洲所有醫師中，首推耆域醫王（頻婆娑羅王之子，精通醫藥，是當時僧團的特約醫師）第一。但假設三千大千世界所有眾生都像耆域那樣精通醫術，若有人問「心中結使、煩惱、邪見、疑、悔」這些心病該用什麼藥醫治，大家竟連答都答不出來，何況能治療！印順導師點出：結使煩惱是生死的根本，有煩惱就有業，有業就有生死苦報，所以解脫生死重病，要從根治煩惱下手；但世間醫師即使懂心理治療，對煩惱的頭數、力用，根本不明白病源，當然無從下藥。菩薩對此有深刻的認定，因而發願：「我終不以世藥為足，我當求習出世智藥，亦修一切善根福德」——福慧雙修，才能自他兩利。

己一 標法性空以觀心：先觀一切法空，再返觀心空

印順導師在此比較唯識與中觀兩家的修證次第：唯識宗先觀所取（境）空，再觀能取（心）空，然後趣入現證，以「唯識無境」為方便；中觀宗則先泛觀一切法空，再觀能觀的心也空，以「緣生無性空」為方便。本經更順於中觀的修法。

經文先教人了知「諸法從緣合生」——一切法依眾緣而有，離開因緣就什麼都不能存在；由此深信「一切法無我、無人、亦無眾生、壽命、知、見、作、受」（我的異名，前面我空觀已說我、人、眾生、壽命，這裡再補四名：知、見、作、受，都是外道凡夫對自我的種種指稱），也就「信解通達無我我所」——無我是我空，無我所是法空。上來是泛觀一切法空；接著才進一步返觀「這能觀的心相也空」——因為一般人「依識立我」，這是廣觀法空而後反觀我空的修證次第。

教學提示：慧可求心的公案

印順導師在此引了一個著名的禪宗典故：慧可禪師向達摩祖師求安心法，達摩說「將心來，與汝安」——這正是要他返觀自心、勤求心相；慧可推求的結果是「覓心了不可得」，這正是安心解脫的不二法門。這個公案很適合作為帶入下文「求心相」推理的引子。

辛一 求心了不可得：三門推求

經文教人如是求心：「何等是心？」印順導師整理出經文所用的三種推求方式：

推求門內容結論
三毒求心是貪欲嗎？是瞋恚嗎？是愚癡嗎？——若心是貪，瞋癡就不是心；若說可以同時相應而有，又不免貪瞋相違，且相應共有正說明是眾緣和合，而非自性不二的心體從三毒推求，什麼都不能定說是心
三時求心是過去的嗎？過去已滅，即是盡滅；是未來的嗎？未來未生未至；是現在的嗎？現在即生即滅，並無一念安住不動的相《金剛經》：「過去心不可得，現在心不可得，未來心不可得」
三處求心在內？在外？在中間？——在內說不出確切處所，且若心在內怎能了外境；在外又怎能覺了自身；中間只是相待的假名，並不能確指不見有一定法名為心，不見心有自性可得

推求之後，經文再以七句顯示心相不可得：無色（非色根識所得）、無形無對（不如五塵有形有對）、無識（非意根識所了）、無知（非有漏識所知）、無住（非依根住而有所得）、無處（非有一定處所）——連「十方三世一切諸佛」，過去不已見，現在不今見，未來不當見。這裡經文反問「云何當有」，印順導師點出這既是反問（表顯心不可得），也伏著一個大哉問：如果心真的了不可得，那我們現前活生生的心識活動又是怎麼回事？他舉了德山禪師被賣點心婆子一問「三心不可得，你點的是哪個心」而啞口無言的公案，說明「解空不能達有」的誤區。經文的解答是：「但以顛倒想故，心生諸法種種差別」「是心如幻，以憶想分別故，起種種業，受種種身」——心是本性空而如幻的有，並非在不可得之外，另有一個微妙真實的心；由於無始妄想，才造業受果，生死流轉。

辛二 二十四句呵心妄有：漢傳佛教最著名的「罵心」段落

印順導師特別指出，這一大段「呵妄心」的譬喻，共二十四句，是集《阿含經》與毘尼（律藏）中散說呵心譬喻之大成，如來在本經中一一總敘出來——目的是讓眾生知道心的幻妄而不迷惑，知道心的性空而不執著，才能離妄心而現證無分別法性。全部二十四句依內容可分四段：

段落譬喻（心如……）要義
一、明心無常不淨（5句）如風不可捉剎那生滅，迅速無痕
如流水生滅不住長江後浪推前浪，覺得此心早已不是此心
如燈焰眾緣有看似長明，實由眾緣前後相續而非一
如電念念滅一霎即過
如虛空客塵污本淨而為外來客塵所染——「心性本淨，客塵所染」，但不可誤解為別有一清淨真心
二、廣明心的作用（7句）如獼猴貪六欲跳躍不停，貪這貪那不息
如畫師起種種業因緣憑空能畫出種種善惡業
不一定隨逐種種煩惱沒有定性，隨環境轉
如大王一切諸法增上主心是一切法中最有力的主導者
常獨行無二無伴一念之中，無二心能同時
如怨家能與一切苦惱認識不全，煩惱相應
如狂象蹈諸土舍顛倒起來能壞一切善根
三、明顛倒心（3句）如吞鉤苦中生樂想如魚吞鉤，被釣住卻以為美味當前
如夢於無我中生我想顛倒錯亂，以為別人就是自己
如蒼蠅於不淨中起淨想在糞穢上亂飛，顛倒生淨想
四、形容心的邪惡（4句）如惡賊能與種種考掠苦殘酷手法奪取
如惡鬼求人便伺機而入，乘虛作怪
常高下貪恚所壞沒有平等心，因愛憎而變異
如盜賊劫一切善根不加防備即被劫奪
五、明貪五欲的過失（5句）貪色如蛾投火死活不顧
貪聲如久行軍樂勝鼓音習慣成癮
貪香如豬樂臥不淨中越貪越過癮
貪味如小女喫毒果而死王女因貪食珍果、宮人為奪果而終致中毒的故事
貪觸如蠅著油終於溺死油中
「心如大王」「心如畫師」「心常獨行」——印順導師特別點出這三句是佛法中最常被引用的譬喻。其中「心如畫師」一句，正與《華嚴經》「心如工畫師，能畫諸世間」同一血脈；理解這二十四句時，最重要的是印順導師反覆強調的原則：知道心的幻妄而不迷惑，知道心的性空而不執著——不是要否定心的作用（心確實能造業、能起種種相），而是要看清這一切作用都是如幻而無自性的。

辛一 無性即性：從「求心不可得」一路推到「無為」

印順導師特別提醒一個關鍵原則：「無性為自性」有甚深意義——如果在無性之外另求一個自性，把無性與自性對立起來，那就失去了如來說法的善巧方便。經文從「求是心相而不可得」出發，是一條環環相扣的推理鏈：

求心相不可得 → 非過去未來現在（超越時間）→ 出三世 → 非有非無 → 即是不起 → 若不起即是無性 → 若無性即是無生 → 若無生即是無滅 → 若無滅則無所離 → 若無所離則無來無去、無退無生 → 則無行業 → 若無行業，則是無為。

印順導師逐環解釋：「有」一定是因緣和合的有，有時間相可說；既不落時間相，就不能說有，也就非無（有無相對而立，非有則非無）。非有非無，就不落意識、不落言詮，即是「不起」——不起，即是無自性（若有自性，就有現起的可能）；無自性，即是無生（「若說緣生即無生，是中無有生自性」）；無生也就無滅；無滅則無所離；無來無去（生死往來不可得）、無退無生（善惡得失不可得），就沒有行業（業是動作事業）可說；沒有行業，就是業煩惱所不起、生滅所不得的「無為」。

壬一、壬二 聖性相：泯諸相與顯淨德

壬一 泯諸相：從無為一路推到「即是聖性」

經文接著說：「若無為者，則是一切諸聖根本」——無為法是三乘聖者（聲聞四果、緣覺、大地菩薩、佛）的根本，《金剛經》也說「一切賢聖，皆以無為法而有差別」。接著又是一條推理鏈：

無為中無持戒亦無破戒 → 無行亦無非行 → 無心亦無心數法 → 無業亦無業報 → 無苦樂 → 即是聖性。

印順導師解釋：戒有能持所持、持犯的增減相待，但深求一切法真相，這是無性空不可得的——「戒性空，持犯不可得故」；持戒破戒既不可得，即無行、無非行；無行無非行，當然無心、無心所可得；無心心所，就無業、無業報；無業無業報，即無苦樂（超脫了生死繫縛的業報），這就是離繫的聖性。他也點出：聖性、佛性，都只是「無為──法空性」的別名，聖性中沒有優劣、增減、得失，一切平等——「是法平等，無有高下」。

壬二 顯淨德：十四句顯示聖性的清淨德性

在不可說而又不可不說之中，經文以十四句方便顯示聖性的清淨德性：

句內容與理由
一、平等如虛空故——不生不滅，不增不減，不垢不淨
二、無別一切諸法等一味故——如四河入海，同一鹹味
三、遠離離身心相故
四、離一切法隨順涅槃故
五、清淨遠離一切煩惱垢故
六、無我離我我所故
七、無高下從平等生故
八、真諦第一義諦故
九、無盡畢竟不生故
十、常住諸法常如故（《法華經》：「是法住法位，世間相常住」）
十一、安樂涅槃為第一故
十二、清淨離一切相故（約一切相不現說，與第五句約離垢說不同）
十三、無我求我不可得故
十四、真淨從本已來畢竟淨故——不是始染終淨，不是離染成淨

印順導師特別點出最後這一組十四句與常樂我淨四德的異同：內容大致相同（樂、淨、常都出現了），但這裡始終說「無我」而不說「我」——雖然聖性也可以說是「我」（「得自在故」），但為了避免與凡外的真我、常我相混濫，本經保有無我說的特色，不說是我。全篇最後總結：「這就是菩薩的出世智藥。以此自得解脫，也以此解脫眾生。現證聖性，成就一切清淨功德性：聲聞、緣覺、菩薩、佛，只是智證的分滿而已」——聲聞、緣覺、菩薩、佛的差別，只在於對同一聖性證得的深淺圓滿程度不同，並非本質有異。至此，「正說菩薩道已究竟」，全經乙一大科圓滿完成。

深讀講義：建議帶讀節奏

教學目標：這是全經正說菩薩道的壓軸，義理環環相扣，二十四句呵心譬喻又極生動——建議至少分三次帶完，讓學員先感受「心了不可得」的震撼，再享受二十四句譬喻的畫面感，最後跟上從無為到聖性的邏輯推導。
次第涵蓋範圍本次重點可延伸提問
第一次戊一舉喻起說＋己一標法性空＋辛一求心不可得（P171–181）耆域醫王也治不了心病；三毒三時三處求心了不可得；慧可求心公案。「求心了不可得」與「感覺不到心」有什麼不同？我們平常說「靜下心來」，這個「心」跟經文推求的「心」是同一回事嗎？
第二次辛二 二十四句呵心妄有（P181–187）五段二十四句的完整內容——無常不淨、廣明作用、顛倒心、心的邪惡、貪五欲的過失。二十四句譬喻中，哪一句最貼近你自己最近的心念經驗？試著用自己的話重新描述一次。
第三次辛一無為相＋壬一壬二聖性相（P187–196）從求心不可得一路推導到無為；泯諸相顯聖性；十四句顯淨德；「正說菩薩道已究竟」總結。「聲聞、緣覺、菩薩、佛，只是智證的分滿而已」——這句話對我們理解不同修行階段、不同法門之間的關係，帶來什麼樣的視野？

教學抓法

  1. 用慧可求心與德山被問倒的兩個公案開場，先勾起學員的好奇心。
    兩個公案一正一反：慧可因求心不可得而安心，德山卻誤把「不可得」當成「沒有」而語塞——正好標出這一單元最容易誤解的分寸。
  2. 二十四句呵心譬喻，適合做成一份「心念觀察日誌」小卡。
    請學員一週內，每天挑一兩句對照自己當天的心念狀態做記錄，下次共修時分享——這比單純背誦二十四句有效得多。
  3. 從無為到聖性的推理鏈，建議畫成一條長長的骨牌鏈圖。
    每一個「若……則……」畫一張骨牌，一路推倒到「無為」再到「聖性」，讓學員直觀感受這是一條環環相扣、不能跳步的邏輯論證，而不是隨意的联想。
  4. 用「正說菩薩道已究竟」作為總結整個乙一大科（單元3–9）的契機。
    可以請學員回顧從修廣大正行、習甚深中觀到作教化事業，用自己的話重新講一次這條完整的修學路徑——這是檢驗前七個單元學習成效的好方法。
討論提問：
  • 「心性本淨，客塵所染」——印順導師特別提醒不能誤解為「有一個清淨真心」。這句提醒，對我們平常聽到「找回本心」「回歸真我」這類說法，帶來什麼樣的辨正？
  • 二十四句呵心譬喻中，「心如大王」「心如畫師」形容心的強大主導力量；但同時又說心了不可得、無自性——這兩者是否矛盾？如何理解「有作用卻無自性」？
  • 「聲聞、緣覺、菩薩、佛，只是智證的分滿而已」——這句話會不會削弱我們對「發菩提心、行菩薩道」的特別重視？還是恰好相反，是在說明所有修行終究同源同歸？
