煩惱雜染收尾：七種差別總覽

七種差別內容
煩惱發業一切煩惱都能發業，但要「猛利現行」（不是一時失念）加上「分別起」（見道所斷，非任運俱生）才能牽引惡趣業；具足三種因緣——殷重無間視為功德、以之為身語意惡行的依處、斷他善品——決定往惡趣受報。
煩惱相自相（貪瞋等各自的體性）、共相（一切煩惱皆為「不寂靜相」）、差別相（門差別：結、縛、隨眠、隨煩惱、纏等26種異名；轉差別：隨眠轉、所緣轉、現行轉、品差別轉、力無力轉、因果轉、迷行轉七種生起樣態，詳見下方「十八種隨眠」獨立章節）。
煩惱聚煩惱聚集的體性與差別分類。
煩惱斷七科細分（能斷已斷、斷所從、斷頓漸、斷次第、斷多種、斷已相、斷勝利），末科「斷勝利」詳見下方獨立章節。
煩惱所緣十五種所緣差別：具分緣、一分緣、有事緣、無事緣、內緣、外緣、現見緣、不現見緣、自類緣、他類緣、有緣、無有緣、自境緣、他境緣、無境緣。
煩惱現行二十種補特伽羅（人）、二十種煩惱現行、二十種煩惱現行緣的交叉分析。
煩惱結生相續說明煩惱（尤其貪愛、疑、薩迦耶見）如何在臨終時現行，牽引有情投向下一生。

十八種隨眠：煩惱種子的多重面貌

隨眠是煩惱的種子。論主用十八種樣態（八組對比），說明煩惱種子不是鐵板一塊——同樣是「種子」，它可以被削弱、可以增長蔓延、可以完全具足或殘缺、可以容易或難以對治。這套精細分類，是理解「為什麼同一種煩惱在不同人身上輕重不同」的理論基礎。
對組十八隨眠
自他境對隨逐自境隨眠・隨逐他境隨眠
損不損對被損隨眠・不被損隨眠
增不增對隨增隨眠・不隨增隨眠
具不具對具分隨眠・不具分隨眠
害不害對可害隨眠・不可害隨眠
強弱對增上隨眠・平等隨眠・下劣隨眠
覺悟對覺悟隨眠・不覺悟隨眠
生苦對能生多苦隨眠・能生少苦隨眠・不能生苦隨眠

斷煩惱的殊勝利益

超越四受憂苦喜樂，證四禪
→
超越色想有對想證四空定
→
超惡趣苦有學位
→
超越一切苦無學位
五種證得內容
安隱遠離煩惱的恐怖畏懼；阿羅漢是「第一安隱」。
清涼遠離煩惱的熱惱；阿羅漢是「第一清涼」。
樂住自在證得第一現法樂住（如滅盡定），能隨心自在安住聖住（空、無願、無相、滅盡定）、天住（四禪四空定）、梵住（慈悲喜捨）。
所證無退所證得的聖道，行住坐臥中都不會再退轉——這是最關鍵的一點。
自義圓滿自己該做的都已完成，不再有所希求；若是菩薩，還能進一步成就利他行，令天人得利益安樂。
這份「斷煩惱利益清單」，讀起來很像一份具體的解脫體驗報告——不是抽象的「你會很快樂」，而是逐項列出：不會再恐懼、不會再燥熱不安、能夠隨心所欲安住在各種殊勝定境、而且這一切都不會退轉。對修行人而言，這是值得反覆閱讀、當作目標的一段文字。

業雜染開篇：五相圓滿業道

染污心貪瞋癡現前
→
起彼欲樂生起造業的意願
→
於是處鎖定正確對象
→
業現行動身發語去做
→
得究竟完成造業

這「五相」是整部論對「什麼才算是完整的一次造業」最精確的判準——五相缺一，這件事在因果上就不算「圓滿業道」。

缺一不可的四種「闕如」說明
闕無欲樂有染污心、對象正確、也造業成功，但沒有造業的意願（如被指使、被迫）——不算圓滿業道。
闕於是處有染污心、有造業意願、也造業成功，但因顛倒心認錯對象——不算圓滿業道。
闕業現行有染污心、有意願、對象正確，但沒有真正付諸行動——不算圓滿業道。
闕得究竟有染污心、有意願、對象正確、也付諸行動，但沒有真正完成——不算圓滿業道。
舉例來說：若有瞋心也真的殺了人，但完全沒有殺人的意願（比如被逼迫、誤觸機關），就「闕無欲樂」，不算圓滿業道；若殺錯對象（顛倒心認錯人），就「闕於是處」。這套判準對現代人常有的「意外造成傷害算不算罪」的困惑，提供了清晰的佛法判斷框架。

菩薩殺生不犯戒的公案

五相圓滿業道的判準，直接關聯到一個著名的公案——《瑜伽師地論・菩薩地》卷41記載：菩薩見到強盜為了謀財要殺害多人，或要加害大德聖者、造無間業，菩薩心想：如果我不阻止他，他將造無間業、來世受無量苦；我寧可自己因殺他而墮地獄，也不忍讓他造下無間重罪。菩薩於是以憐愍心、慚愧心斷了那個惡人的性命——常柏法師指出，這件事因為菩薩的「染污心」環節被「善心或無記心」取代，所以不但無所違犯菩薩戒，反而生多功德。這是判斷「殺生業道是否圓滿」的五相架構，在最極端的倫理處境下如何被謹慎運用的經典範例。

細部架構：五個值得多停留的地方

一、七種差別是「煩惱百科全書」的完整目錄

發業（誰會造業）、相（長什麼樣子）、聚（怎麼聚在一起）、斷（怎麼斷除）、所緣（緣什麼）、現行（什麼時候發作）、結生相續（如何牽引下一生）——七個角度合起來，構成對「煩惱」這件事最全面的分析，值得讀書會當作觀察自己內心的完整檢核清單。

二、「猛利現行」與「分別起」兩個條件，說明不是所有煩惱都一樣危險

一時失念生起的煩惱，即使聖人偶爾也會有，但因為不夠猛利、也不是見道所斷的分別起煩惱，並不會牽引到三惡道——這對容易因為偶爾的負面情緒而過度自責的人，是重要的安慰：輕微、短暫的煩惱與根深蒂固、猛利的煩惱，在因果上的份量並不相同。

三、斷煩惱的殊勝利益值得抄下來當修行的座右銘

「所證無退」這一項尤其關鍵——很多人擔心修行會退轉，但論主明確指出：聖道一旦證得，不會再退失，這與禪定境界（鈍根者可能退失）是不同層次的東西。

四、五相圓滿業道，是佛法版本的「犯罪構成要件」

染污心、起欲樂、於是處、業現行、得究竟，五個環節環環相扣，任何一環缺失，都不構成「圓滿業道」——這套精密的分析架構，其實與現代法律區分故意、過失、既遂、未遂的邏輯有異曲同工之妙。

五、菩薩殺生的公案，示範了戒律不是死板的規則

論主用「染污心是否現前」作為關鍵判準，而不是單看外在行為本身——這提醒我們，佛法對善惡的判斷，核心永遠在於起心動念而非表面的行為，但這絕不是說一般人可以援引此例合理化自己的惡行，經文特別強調這是菩薩以甚深慈悲與智慧才能運用的特殊情況。

深讀講義：卷五十九可分三段帶讀

深讀目標：本卷前段收尾煩惱雜染，後段開啟業雜染；建議把「斷勝利」與「五相圓滿業道」當成重點，十八隨眠與七種差別的細節用總覽表快速帶過。

甲、煩惱雜染收尾：七種差別與十八隨眠（約30–35分鐘）

時間內容方法
0–20七種差別總覽用表格快速帶過發業、相、聚、斷、所緣、現行、結生相續七項，不逐項細講。
20–35十八隨眠用八組對比帶讀，強調煩惱種子有強弱、增減、具缺之別。

乙、斷煩惱的殊勝利益（約20分鐘）

時間內容方法
0–20五種證得功德逐項帶讀安隱、清涼、樂住自在、所證無退、自義圓滿，強調這是具體可以嚮往的目標。

丙、業雜染開篇：五相圓滿業道與菩薩公案（約30分鐘，重點段落）

時間內容方法
0–15五相圓滿業道用殺生為例，帶讀染污心、起欲樂、於是處、業現行、得究竟五個環節。
15–30四種闕如與菩薩公案用「意外傷人算不算罪」的現代情境帶讀四種闕如，再講菩薩殺盜賊的公案，說明起心動念才是關鍵判準。

讀書會抓法

  1. 七種差別可以當作觀察自己煩惱的完整檢核清單。
    試著用發業、相、聚、斷、所緣、現行、結生相續七個角度，分析自己最近的一次煩惱。
  2. 「猛利現行」與「分別起」提醒我們不必為偶爾的負面情緒過度自責。
    真正危險的是根深蒂固、被自己合理化的煩惱。
  3. 斷煩惱的殊勝利益清單值得抄下來。
    尤其「所證無退」這一項，可以當作修行信心的支柱。
  4. 五相圓滿業道可以用來重新檢視自己的行為。
    是不是常常「闕於是處」（對象搞錯了才後悔）或「闕無欲樂」（不是真心想做但還是做了）。
  5. 菩薩殺生公案提醒我們：佛法判斷善惡看的是起心動念。
    但這需要甚深的智慧與慈悲才能正確運用，不是一般人可以援引的藉口。
讀書會提問：
  • 「所證無退」對我們害怕修行退轉的焦慮，有什麼具體的安慰作用？
  • 五相圓滿業道的判準，跟現代法律區分故意、過失、既遂、未遂有什麼異同？
  • 菩薩殺盜賊的公案讓你感到不安還是佩服？為什麼佛法要收錄這麼極端的例子？
