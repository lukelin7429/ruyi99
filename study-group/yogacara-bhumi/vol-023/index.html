根律儀：守護六根的五種方法

「根律儀」就是守護眼耳鼻舌身意六根。六根是各種境界進出的門戶，境界一進來，心就容易在可愛境起貪、不可愛境起瞋、中庸境起癡。根律儀要做的，就是在這個「進出口」把關。本論用五種方法（加上第六「修意根律儀」）來守：

方法要義
一、密護根門總說。六根是境界出入之門，緊密地把守這道門，使煩惱未生不生、已生能斷——這是總綱，底下四種是具體做法。
二、防守正念正念本身有「防惡」的功能。由聞思修的力量獲得正念，再時時串習、使正念不忘失、不失壞——先要有正念這個守門人。
三、常委正念把正念保持得「恆常無間、詳細周備」——不是偶爾提起，而是綿密不斷地現前。
四、念防護意用正念防止第六意識起「不如理作意」——守住那個會把中性境界加油添醋的念頭。
五、行平等位讓心安住在「善捨」或「無記捨」——貪瞋癡一起，馬上棄捨，使心平等、情緒不高高低低。棄捨有兩種：有智慧的善捨、與不隨順染心的無記捨。

這五種其實是一套機制：密護根門是目標，防守正念、常委正念是「守門人」（正念）的品質，念防護意是守住起念的關口，行平等位是煩惱真的冒出來時的即時處理。守根不是把眼睛閉起來不看，而是看了、聽了，心不被牽著走。

不取相・不取隨好

五種方法之外，根律儀最核心的一個實修口訣，是「於色聲香味觸法，不取其相、不取隨好」——這是整個守根的關鍵動作。

不取其相「取相」是於眼識所行的色境中，執取那個境界的整體相狀（例如看到一個人，就抓住「這是美的／醜的」這個總相）。不取其相，就是不在第一時間對整體下貪瞋的判斷。
不取隨好「隨好」是進一步執取境界的種種細部好相（美在哪裡、好在哪裡，一個細節一個細節地回味）。不取隨好，就是不讓心在細節上流連、越描越貪。
為什麼要「不取相、不取隨好」？因為一旦執取了整體相、又追逐細部好，就會「依彼六塵而生諸惡不善尋思，令心流漏」——貪瞋癡的尋思就順著這個相流出來，心就漏了。守根的功夫，全在境界剛接觸的那一剎那：看只是看、聽只是聽，不加上「這是好的、我要」的第二念。這正是南傳「見以見為量」（diṭṭhe diṭṭhamattaṃ）的同一個心法。

於食知量：四不四為

第六種資糧「於食知量」，把修行的功夫用在最日常、最容易放逸的一件事上——吃飯。它的總綱是：以「正思擇」（智慧觀察）來吃，具體展開成「四不、四為」：

四不——不為這些而吃
  1. 不為倡蕩：不為欲貪婬逸所引、令諸根掉舉躁擾而吃（不是為了尋歡助興）
  2. 不為憍逸：不為無病憍、少壯憍、長壽憍而吃（不是為了逞強使氣）
  3. 不為飾好：不為沐浴、理髮、塗香等打扮而吃
  4. 不為端嚴：不為以華服華鬘莊飾身相、好看而吃
四為——只為這些而吃
  1. 為身安住：為維持這個色身安穩存活
  2. 為暫支持：為暫時支撐身體（不是貪養）
  3. 為除飢渴：為解除飢渴的逼迫
  4. 為攝梵行：為攝持清淨的梵行、有力氣修道

這「四不四為」正是佛門「食存五觀」的根本精神，也對應南傳每餐前的省思——「不為嬉戲、不為驕慢、不為裝飾、不為莊嚴，只為此身住立、除飢渴、助梵行而受食」。吃飯不是為了享受口腹，而是為身體加油、好繼續修行。

觀食三過患・如食子肉想

正思擇——先看見食物的三種過患

「正思擇食」的具體做法，是吃之前先以智慧觀察「段食」（吃進肚子的食物）的三種過患，見過患後深生厭惡，然後才吞咽：

三種過患觀什麼
受用種類過患再精美的食物，一進口中被牙齒咀嚼、唾液浸爛，「先前悅意妙相一切皆捨，轉成可惡穢相，狀如變吐」——若正念這個嚼碎後的樣子，連對還沒入口的美食都提不起貪了。
變異種類過患飲食會引發種種疾病（癰、痤等）；食物在身中變異，也帶來種種身體的過患。
追求種類過患為了求得飲食，要付出種種追求、經營的辛勞與過患。

如食子肉想——體念施主的恩

為求出離而吃，應「如食子肉想」而食：施主是「甚大艱難積集財寶」，經歷種種追求的辛苦，出於悲愍、為求勝果，「如割皮肉及以刺血」而布施。所以行者受食應當這樣想：我受這食，要知量而食（不過飽、不過少，剛好堪能修行），如佛所說安住在法上受用，不顛倒受施主食；還要用這力氣修行、弘法利生，使施主得最殊勝的果報——這才是報施主恩。「如食子肉想」出自《雜阿含》——像走投無路的父母在曠野中，為活命而含淚吃下亡兒之肉，絕無一絲貪著，只為活著走出曠野。受食也當如此：只為活命修道，不為貪味。

有艱難存養 vs 無艱難存養

有艱難存養（該避免的吃法）吃得讓身心負擔更重：或多食令「身沈重、無所堪能，不任修斷」；或令「心遲鈍、不速得定」；或令「入出息來往艱難」；或令「心數為惛沈睡眠所纏擾」。吃太飽反而修不下去。
無艱難存養（該追求的吃法）吃得剛剛好，讓身體輕安、堪能修行——不增飢羸困苦，也不因貪味而染著耽嗜。以正法追求、知量而食，身心才能保持在最能用功的狀態。

細部架構：讀懂守根與知食的幾個關鍵

一、戒、根律儀、食知量是一組「由外而內」的守護

戒守身口的行為（不犯），根律儀守六根接觸的剎那（不漏），食知量守最日常的口腹之欲（不放縱）。三者一層比一層細、一層比一層貼近生活——把這三道基本功做好，日常的散亂與粗重就先降下來一大截，才談得上修定。

二、守根的關鍵在「第一念」——不取相、不取隨好

根律儀不是要人閉目塞聽，而是在境界剛接觸的剎那，不對它的整體相與細部好起貪瞋。守得住這「第一念」，後面的惡不善尋思就流不出來。這是最省力、也最難的地方——功夫全在當下一剎那。

三、於食知量，把「觀」用在最容易放逸的地方

吃飯是每天必做、最容易貪著的事。本論偏偏在這裡下手，用「觀三過患」把貪味的心澆熄、用「四不四為」把吃的動機擺正、用「如食子肉想」把感恩與慚愧提起。這說明修行不在別處，就在最日常的一飯一食之間。

四、「有艱難／無艱難存養」——吃太飽是修行的敵人

本論很務實地指出：吃太飽會讓身沈重、心遲鈍、呼吸不順、昏沉想睡，全都是修定的障礙。知量而食不是苦行，而是把身體維持在「最能用功」的狀態——這是非常實際的修行智慧。

讀書會抓法

  1. 根律儀不是閉目塞聽，而是看了聽了、心不被牽著走。
    回想今天，有沒有哪一個眼神或聲音，一接觸就勾起了你的貪或瞋？當時若能「不取相、不取隨好」，會不會不同？
  2. 守根的功夫全在「第一念」——不對境界的整體相與細部好起判斷。
    試著練習一次「看只是看、聽只是聽」，不加上第二念的「我喜歡／我討厭」，感受一下有什麼不同。
  3. 於食知量的總綱是「四不四為」——不為玩樂憍慢裝扮好看，只為身安住、除飢渴、助梵行而吃。
    下一餐吃飯前，試著問自己：我這一口，是為了什麼而吃？
  4. 「如食子肉想」把受食變成一場感恩與慚愧的功課。
    想想你桌上的食物，背後有多少人的辛勞？帶著這份體念吃飯，心會不會更收攝？
  5. 吃太飽會讓身沈重、心遲鈍、昏沉想睡——是修行的敵人。
    觀察自己：是不是常常吃到「無艱難存養」的分量之外，反而讓自己更懶散？
讀書會提問：
  • 「不取相、不取隨好」在資訊爆炸、影音刺激無所不在的今天，特別難做。你覺得現代人要守根，最大的挑戰在哪裡？
  • 本論把「吃飯」提升成一項修行功課（觀三過患、如食子肉想）。這對你平常的用餐習慣，有什麼提醒？
  • 戒、根律儀、食知量都還只是「資糧」、還沒開始修定。你覺得為什麼要花這麼多功夫先調日常生活，而不是直接教打坐？
