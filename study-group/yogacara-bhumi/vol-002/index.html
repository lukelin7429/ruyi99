瑜伽師地論卷二導讀

用圖表掌握卷二：一切種子識、三種習氣、三界隨逐、名色增長，以及世間離欲與出世間離欲的差別。

本卷圖表總覽圖講記骨架關係圖判讀表細部架構讀書會抓法名相索引

總覽圖：卷二的主軸是「種子如何支持生死」

一切種子識阿賴耶識攝藏種子
→
三種習氣名言、我執、有支
→
受生自體名色、六處、身心果報
→
三界五趣流轉種子隨逐，遇緣現行
卷二的主軸：不是只說「阿賴耶識藏種子」，而是說明有情為何一再受生、為何世間禪定不能究竟解脫、為何必須斷我愛種子。

講記骨架：常柏法師如何拆卷二

意地中一切種子識
  • 種類分別
    • 具一切種：有涅槃法者具足有漏、無漏種子
    • 闕菩提種：不般涅槃法者闕三種菩提種子
  • 隨逐分別
    • 欲界自體中有色、無色界種子
    • 色界自體中有欲、無色界種子
    • 無色界自體中有欲、色界種子
  • 和合增長
    • 羯羅藍漸增長時，名與色平等增長
    • 地水火風各有業用，使根身成長
  • 習氣與受果
    • 名言習氣是生自體最勝因
    • 業習氣使果報勝劣差別
    • 我執習氣令我愛與我慢相續
  • 所依分別
    • 煩惱品種子名麤重、隨眠
    • 異熟無記種子名麤重，不名隨眠
    • 善法種子有堪能性，不名麤重

關係圖一：三種習氣如何支撐流轉

名言習氣親生有為法與身心自體
我執習氣生起我、我所、自他差別
有支習氣招感三界異熟果報
↓
阿賴耶識相續種子受熏、隨逐、待緣
→
一一自體中皆有一切自體種子五趣流轉的根本機制

關係圖二：世間離欲 vs 出世間離欲

世間離欲

依欣上厭下修禪定，暫時離下地欲，可生色界、無色界；但我愛未斷，果報盡後仍回流轉。

出世間離欲

依無我理證現觀，斷除我愛種子；若於一處離我愛欲，即於一切處得離欲。

判讀表：卷二重點名相不要混在一起

概念常柏法師講記中的作用讀者要避免的誤解
一切種子識攝藏有漏、無漏種子，也是種子受熏、隨逐、取果的依止。不要把它當作靈魂或固定自我。
菩提種子三種菩提：聲聞、獨覺、大菩提；關係到能否入涅槃。不要只當成道德善根，它牽涉出世道可能性。
隨逐受生某界時，其他界地種子仍在自體中等待因緣。不要以為換到上界就完全脫離下界種子。
名言習氣親生眼等六處自體，是生自體的最勝因。不要只把業力當作生命的唯一因。
麤重無堪能性，使有情不能自在修善、證道。不要只理解成粗重的身體感，這是種子層面的障礙。
轉依聲聞轉令緣闕；菩薩轉得內緣自在，起變化身利他。不要把聲聞與菩薩轉依混成同一種結果。

細部架構：卷二要畫成「種子流轉系統」

一、種子從潛伏到流轉的流程

熏習現行熏成種子
→
隨逐種子隨自體保存
→
待緣遇生緣、業緣、愛取滋潤
→
現行取果受三界五趣異熟
→
再熏習流轉相續不斷

二、三種習氣的分工

習氣親近功能它造成什麼修行如何對治
名言習氣親生有為法各別種子，是生自體的重要因。讓身心世界、六處、分別語言相續現前。觀察名言所安立的世界不是真實自性。
我執習氣虛妄執我、我所的種子。自他差別、我慢、我愛、保護自我的反應。修無我觀，破我與我所的根本錯認。
有支習氣招感三界異熟的業種。善惡趣、貴賤、壽夭、資具等果報差別。斷惡修善只是第一步，還要出離有漏生死。

三、三界隨逐圖：為何上生不等於解脫

現在受生仍隨逐的種子常柏法師要讀者看見什麼
欲界色界、無色界種子仍在。欲界眾生仍能修定上生，表示自體中具上地種子。
色界欲界、無色界種子仍在。色界壽盡或定力因緣改變，仍可能再受其他界果報。
無色界欲界、色界種子仍在。非想非非想處壽盡後仍會再受下地果報。
聖者出離依無我現觀斷我愛種子。真正關鍵不是住在哪一界，而是我愛種子是否斷。

四、受生自體：名色增長的層次

羯羅藍入胎最初位，名色和合開始增長。
名受、想、行、識等心理面向。
色父母不淨和合所成，四大種功能逐漸顯現。
地界令依止造色增廣。
水界攝持不散。
火界成熟堅硬。
風界分別支節，各安其所。
依止圓滿諸根依處分明，能為心心所依止。

五、麤重判讀表

種子品類是否名麤重是否名隨眠理由
煩惱品是是無堪能性，且能生煩惱現行，隨逐有情增長過失。
異熟無記品是否無堪能性，但不直接生煩惱。
善法品否否有堪能性，能違背煩惱、成就善法。

讀書會抓法

  1. 先畫出「種子流轉圈」。
    阿賴耶識藏種子，種子遇緣現行，現行又熏種子，若沒有無我觀斷我愛，流轉不止。
  2. 分清三種習氣。
    名言習氣處理身心世界如何生起；我執習氣處理「我」如何相續；有支習氣處理到哪裡受果。
  3. 把「三界隨逐」變成警策。
    即使生到色無色界，下界種子與我愛未斷，仍不究竟。
  4. 用卷二連回修行。
    卷二不是玄學，是在說為何需要戒定慧、無我觀與淨土方便。
讀書會提問：
  • 「一一自體中皆有一切自體種子」為何令人警覺？
  • 如果業不是生自體唯一最勝因，那名言戲論在日常生活中如何觀察？
  • 世間禪定與佛法解脫的差別，如何用現代語言講清楚？
